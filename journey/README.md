# Journey — Live Diary

This folder holds the daily entries of a 30-day public attempt
to send the manifesto to Andrej Karpathy.

Each file `day-XXX.md` is one day's reflection.

## Why This Folder Exists

The manifesto argues that **process matters more than result** and that
**time accumulated is identity itself**. This folder is the live evidence
of those claims.

Whatever happens — whether Karpathy ever responds, whether the campaign
"works" in any conventional sense — these 30 entries will exist as a
record of one human attempting something honestly, in public, with AI
as a thinking partner.

## How to Read

- **Chronologically**: start with `day-001.md` and read forward
- **By theme**: each entry has tags (`#diagnosis`, `#meta`, `#failure`, etc.)
- **Most recent**: pick the highest-numbered file

## How to Contribute

If this work resonates and you want to start your own parallel journey,
you may fork this repository and create your own `journey/` folder.

If you do, please open an issue here labeled `parallel-journey` so others
can find your work. The chain propagates.
