# Contributing to LifeBite

This repository welcomes:

## Translations
Currently available: Korean (full), English (full), French (one-pager).
For other languages, open a PR with a translated file in `manifesto/`.

## Critiques
Sharp disagreements help refine the work. Open an issue labeled `critique`.
Honest reasoning is more valuable than agreement.

## Cell Reports
If you start a small "Context Family" experiment (see `manifesto/en.md` Part 6.3),
share what you learn. Open an issue labeled `cell-report`.

## Extensions
- Academic refinements of LifeBite, synergy, time-inverted identity
- Tool prototypes for the Sovereign Filter
- Policy drafts for relational-unit recognition
- Educational program designs for "teaching the gap"

Open a PR with your contribution in the appropriate folder.

## References
If your work is cited unfairly, or your tradition is missing from the
philosophical lineage section of the README, please open an issue.
Special invitation to traditions underrepresented in current Western
discourse (African, Indigenous, South Asian, etc.).

## How to Open a PR
1. Fork this repository
2. Make your changes in your fork
3. Submit a Pull Request with a clear description
4. Be willing to discuss

## How to Open an Issue
Use one of these labels:
- `critique` — disagreement with the work
- `cell-report` — your experiment report
- `translation` — translation work
- `reference` — pointing to related work
- `bug` — broken link, typo, file structure issue

## A Note on Tone
This is a small repository. Discussions here should reflect the
manifesto's principles: bubble-free, honest, contextual, with respect
for the time of other contributors.

No promotional content. No aggressive engagement. No surface signals.

Thank you for being here.
